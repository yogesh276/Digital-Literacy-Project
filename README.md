Project Overview:
This project is a Digital Literacy Portfolio developed as part of the CSE0001 – Digital Literacy course at VIT Bhopal University. The objective is to build essential digital skills required for students to effectively navigate the online world, maintain a professional presence, communicate responsibly, and stay aware of cyber threats.

As a Student Digital Ambassador, I completed five practical tasks aligned with different modules of digital literacy. Each task focuses on real-world applications such as creating online profiles, using collaboration tools, writing professional emails, and understanding cyber safety.

All project work, including screenshots, documents, and reflections, is organized in this repository
